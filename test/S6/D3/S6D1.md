## Creando el molde fantasma ( el modulo `abc`)
- `abc` = Abstract Base Clasess
- Debemos marcar al menos uno de sus metodos con el decorador `@abstractmethod`
- un metodo abstracto es un metodo vacio. no tiene codigo por dentro. usamos la palabra `pass`

```python

from abc import ABC, abstractmethod

class Documento(ABC):
    def __init__(self, titulo):
        self.titulo = titulo
    
    @abstractmethod
    def exportar(self):
        pass
    def mostrar_titulo(self): # Opcional: los padres abstracion pueden tener metodos normales
        print(f'Documentos: {self.titulo}')

try:
    doc_invalido = Documento('Mi archivo')
except:
    print(f'[BLOQUEO DE SEGURIDAD])

```

## Propiedades Abstractas
ponemos `@property` siguiendo de `@abstractmethod`
*"Estas Obligado a crear un Getter para esta propiedad"*

```python
from abc import ABC, abstractmethod

class VehiculoComercial(ABC):
    @property
    @abstractmethod
    def tarifa_km(self):
        pass

    def calcular_viaje(self, kilometro):
        total = kilometro * self.tarifa_km
        print(f'Costo del viaje {total}')

class Taxi(VehiculoComercial):
    @property
    def tarifa_km(self):
        return 500

mi_taxi = Taxi()
mi_taxi.calcular_viaje(10)

```
