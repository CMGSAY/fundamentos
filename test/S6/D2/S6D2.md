```python
    self.monto = monto
  def __add__(self, otra_billetera):
    suma_total = self.monto + otra_billetera.monto

    #return suma total
    return Billetera(nuevo_propietario, suma_total)
  
  def __gt__(self, otra_billetera):
    return self.monto > otra_billetera.monto
```

## La bodega de Herramientas (Imporatando Modulos)
### forma 1: Importando la caja completa
- Vamos a usar la palabra reservada `Import` seguida el nombre dela caja
- Esto trae la caja entera a nuesta mesa. Pasa usar una herramienta que esta adentro

```python
import math

raiz = math.sqrt(25)

print(f'La raiz cuadrada es: {raiz})

```


### forma 2: Importando la herramienta especifica

### Forma 3: Import con un apodo (Alias)
- Python nos permite ponerle un apodo con `as` (como).

```python
import datetime as dt

fecha_hoy dt.date.today()
print(f'la fecha de hoy es: {fecha_hoy}')
```
