# Funciones o metodos
## sintaxia
para crear una funcion vamos a crear una palabra reservada `def` viene de una palabra de **define** en ingles. le asiganamos un nombre. 
abrimos y cerramos con parentesis  `( )` y terminamos con `:`

ocupamos la identacion para identificar que le pertenece a la funcion.


```python

def saludo():
    print('Hola como estas?')
```

```python

def menu():
    print('1. Saludar')
    print('2.Despedirse')
    print('3. Salir')
    
    opcion = input('selccione una opcion.')
    
    while opcion != '3':
        match opcion:
            case '1':
                saludo()
            case '2':
                despedida()
            case '3':
                print('Saliendo... ')
            case _:
                print('Opcion no valida')
        opcion = input('selccione una opcion.')

menu()

```
# Parametros en una funcion
caracteristicas o valores dentro de las funciones

```python
def menu():
    print('1. Saludar: ')
    print('2.Despedirse: ')
    print('3. Salir')
    
    opcion = input('selccione una opcion: ')
    nombre = input('Ingrese su nombre: ')
    
    while opcion != '3':
        match opcion:
            case '1':
                saludo(nombre)
            case '2':
                despedida(nombre)
            case '3':
                print('Saliendo... ')
            case _:
                print('Opcion no valida')
        opcion = input('selccione una opcion.')

menu()
```
