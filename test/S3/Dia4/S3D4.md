# Estructura de datos
## Diccionarios
Se usa lo que llama "clave y valor"

le ponemos una etiqueta con nombre (clave) a la caja. y desntro guardamos la informacion (valir).

```
clave : valor
```
**Sintaxis** A diferenia de las listas que usan corchetas `[]`
se utilizan las llaves `{}`

``` 
producto = {
    'nombre' = 'arroz',
    'precio' = 1200.00,
    'cantidad' = 5

}
```
un diccionario es ideal cuanod queremos que cada dato tenfa una etiqueta clara.

# acceder a valor dentro de un diccionario 

```python

producto = {
    'nombre': 'arroz',
    'precio':  1200.00,
    'cantidad':  5
}

print(producto['nombre'])
print(producto['precio'])
print(producto['cantidad'])

total = producto['cantidad'] * producto['precio']
print(f'El total de los productos son: {total}')

print(producto)
```
