
## condicional `else`
NO LLEVA NUNCA UN ACONDICION al lado!
* es el basurero que recoge cualquier cosa que haya cumplido con la regla del `if`

### ejemplo
```python

dinero  = float(input ('Dinero del cliente: '))
precio = 1200
if dinero >= precio:
    print('Puede comprar el producto')
else:
    print('no puede comprar el producto')
```

### condicional: switch (match/case)
`match/case`
```python
match valor:
    case patron1:
        # acciones
    case patron2:
        # acciones
    case _:
        # acciones por defecto

```

## ejemplo
```python
opcion input('Selecciones una opcion (1 - 3): ')

match opcion:
    case '1':
        print('Registro de producto')
    case '2':
        print('Mostrando inventario... ')
    case '3':
        print('saliendo')
    case _:
        print('Opcion invalida')

```
