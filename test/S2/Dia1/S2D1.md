# Tipos de datos
1. String: Es de tipo texto. cualquier caracter(letras, numeros o simbolos) que este encerrado entre comillas " " o ' '.
2. Entero, (integer) (`int`): números enteros, si decimales.
3. flotantes. (`float`): números con decimales
4. booleanos(`bool`): solo tiene dos opciones: `True` (Verdadero) y `False`(Falso).
