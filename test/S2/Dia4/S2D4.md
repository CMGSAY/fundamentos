# Tema tareas repetitivas (for, while)

## while (Mientras)
El bucle `while` repite un bloque de codigo *Mientras* una condicion sea Verdadera
(`True`). En el instante en que la condicion se vuelva Falsa(`False`), el ciclo termina.

### Sintaxis
```python
while condicion :
    # accion repetitiva que quiero que se suceda

```

# Ejemplo
```python
contador = 1

while contador <= 3:
    print("Hola soy e ciclo while")
    valor = int(input('ingrese un número pequeño de aumento"))
    contador = contador + valor
```

#Ejemplo 2
```python
claveCorrecta = '1234'

intento = ''

while intento != claveCorrecta:
    intento = input('Ingrese la clave: ')

print('Clave correcta¡ Acceso concedido. ')
```

## ciclo `for`
`range(cantidad)` 

## forma1: range(cantidad)
# Ejemplo:
```python
# 0, 1, 2, 3, 4 
# o ..... n - 1 
for index in range(5):
    print(index)

```

## forma 2: range(inicio, cantidad)
# Ejemplo:
```python
#  1, 2, 3, 4 
# 1 ..... n - 1 
for index in range(1, 5):
    print(index)

```
## forma 3: range(inicio, cantidad, incremento)
# Ejemplo:
```python
#  1, 3, no el 5 
# 1 ..... n - 1 
for index in range(1, 5, 2):
    print(index)

```
